# Sequential Pattern Mining on Online Retail Data

Dự án thực nghiệm Khai phá Mẫu tuần tự (Sequential Pattern Mining) thuộc học phần Khai phá Dữ liệu - **Nhóm 8**.

Dự án tập trung triển khai, so sánh và đánh giá hiệu năng của các thuật toán khả mở (GSP, PrefixSpan) dựa trên lý thuyết Mục 5.4 - sách *Data Mining: Concepts and Techniques (4th Edition)*.
